First release of `frame-master-plugin-mdx-to-js-loader`, a Frame Master plugin that enables MDX support in your projects.

### Features

- **MDX Compilation** - Compile `.mdx` files to JavaScript using `@mdx-js/mdx`
- **React JSX Support** - Built-in React JSX import source configuration
- **Relative Import Resolution** - Properly resolves relative imports from MDX files using `VFile`
- **Customizable MDX Options** - Pass custom `@mdx-js/mdx` compile options via the plugin configuration
- **Runtime & Build Support** - Works seamlessly in both development (HMR) and production builds

### Usage

```typescript
// frame-master.config.ts
import mdxLoader from "frame-master-plugin-mdx-to-js-loader";

export default {
  plugins: [
    mdxLoader({
      // Optional: pass custom MDX compile options
      mdxOptions: {
        // remarkPlugins: [...],
        // rehypePlugins: [...],
      },
    }),
  ],
} satisfies FrameMasterConfig;
```

### Requirements

- Frame Master `^2.1.1`
